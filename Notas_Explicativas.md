# Notas Explicativas 
A solução foi desenvolvida em PHP, utilizando uma estrutura simples para organização do código e execução via linha de comando. O programa realiza a leitura do arquivo input.csv, enriquece os dados dos municípios utilizando a API de localidades do IBGE e gera o arquivo resultado.csv com os campos e status definidos no enunciado.<br>
Para lidar com diferenças de escrita e erros de digitação nos nomes dos municípios, foi aplicada uma normalização de texto combinada com matching aproximado, permitindo identificar corretamente entradas inconsistentes. Os dados do IBGE são armazenados em cache local para evitar chamadas repetidas à API e melhorar a performance.<br>
Após o processamento, são calculadas as estatísticas solicitadas, considerando apenas os registros com status OK. Essas estatísticas são então enviadas para a Edge Function de correção, utilizando autenticação via Bearer Token (JWT), e o score retornado é exibido no console.<br>
Durante a execução em ambiente Windows, foi aplicado um ajuste na verificação SSL para permitir as chamadas HTTPS via cURL. Em um cenário de produção, a verificação de certificados seria mantida com a configuração adequada do CA bundle.
