# Teste Nasajon

## Como Rodar o Projeto

### Pré-requisitos
- PHP 7.4 ou superior

### Instalação

1. Clone o repositório:
2. logo em seguida renomei o arquivo ```.env.exemplo``` para ```.env``` e adicione seu token

logo em seguida no terminal rode o seguinte comando para realizar teste

```cmd
php public/index.php --no-subimit 
```

caso queira enviar o resultado aplique esse comando 

```cmd
php public/index.php
```
